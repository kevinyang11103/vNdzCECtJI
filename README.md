# 学科竞赛管理系统 java910

## 项目概述

学科竞赛管理系统是一款集赛事发布、报名、管理于一体的在线平台。系统采用前后端分离的技术架构，后端基于SSM框架，前端采用Vue.js与HTML5技术。

## 技术栈

### 后端技术

- SSM（Spring + SpringMVC + MyBatis）

### 前端技术

- Vue.js
- HTML5

## 运行环境

- 开发工具：IDEA
- 数据库：MySQL 5.7
- 运行方式：Tomcat

## 系统角色

系统包含以下角色：

- 超级管理员
- 学生用户
- 教师用户

## 功能模块

系统主要包含以下功能模块：

- 赛事发布
- 报名参加
- 成绩管理
- 用户管理

## 部署步骤

1. 准备运行环境：安装配置Tomcat、MySQL。
2. 创建数据库，导入数据表结构。
3. 部署后端代码至Tomcat。
4. 部署前端代码至Tomcat。

## 目录结构

```
项目根目录/
├── backend/    （后端代码目录）
│   ├── src/
│   ├── resources/
│   └── pom.xml
└── frontend/   （前端代码目录）
    ├── src/
    ├── public/
    └── package.json
```

## 核心亮点

- 前后端分离，易于维护与扩展。
- 系统功能模块齐全，满足学科竞赛管理需求。
- 支持多种角色，权限分明。
- 高度可定制，可根据实际需求调整功能模块。
